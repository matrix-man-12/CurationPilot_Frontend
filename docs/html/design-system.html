Frontend Design System

Theme: Warm Pastels
Updated: June 15, 2026

Color Palette (Warm Pastels)

The interface uses a curated, warm pastel-toned color palette designed to feel premium, lightweight, and organic. Standard pure blues or reds are avoided in favor of HSL-derived earthy pigments.

--color-surface #F9F5F0 Page canvas background
--color-surface-raised #FFFFFF Cards, forms, chat box
--color-primary #E8A87C Peach Coral primary links
--color-primary-soft #FEF0E7 User speech bubble highlight
--color-text #3D3229 Warm dark brown body text
--color-text-muted #8C7E73 Secondary labels & subtext
--color-success #7DB5A0 Sage Mint success states
--color-warning #E4B87C Soft gold warning/running
--color-error #D98B8B Soft rose error blocks

Typography & Scaling

Typography is driven by Google Fonts: Inter for user interface text and headings, and JetBrains Mono for parameters, logs, and codes.

CSS Variable Token Calculated Value Equivalent Line Height Standard Usage
--text-xs 0.75rem (12px) 1rem Timestamps, metadata labels, tags.
--text-sm 0.875rem (14px) 1.25rem Secondary copy, labels, help text.
--text-base 1rem (16px) 1.5rem Primary body text, form inputs, button text.
--text-lg 1.125rem (18px) 1.75rem Log box headers, section dividers, card titles.
--text-xl 1.5rem (24px) 2rem Page headers, modal headings.

Spacing Scale

A mathematical 4px-base spacing scale handles all margins, paddings, and flex/grid gaps:

Spacing Token Rem Value Pixel Equivalent Common Layout Application
--space-1 0.25rem 4px Inner element margins, small bullet spacing.
--space-2 0.5rem 8px Gap inside controls, padding for inputs.
--space-3 0.75rem 12px Flex alignments, bubble message gaps.
--space-4 1rem 16px Padding for simple buttons, margins on labels.
--space-5 1.5rem 24px Padding for small cards, gap between sections.
--space-6 2rem 32px Padding for large panels, layout columns gap.
--space-8 3rem 48px Hero header spacing, page body margins.

Borders, Shadows & Radius

Components utilize gentle shadows with low opacity to stand out from the page background, avoiding harsh lines.

  • Border Radius:
    • --radius-sm: 4px — small tags, badges.
    • --radius-md: 8px — inputs, dropdown selections, buttons.
    • --radius-lg: 12px — cards, parameters grid boxes.
    • --radius-xl: 16px — speech bubbles.
  • Shadows:
    • --shadow-sm: 0 1px 2px rgba(61,50,41,0.05) — simple raised buttons.
    • --shadow-md: 0 4px 12px rgba(61,50,41,0.08) — sidebar elements, dropdown sheets.
    • --shadow-lg: 0 8px 24px rgba(61,50,41,0.12) — modal dialog overlays.
